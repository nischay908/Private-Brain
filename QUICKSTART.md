# Quick Start Guide - ThinkLocal

## What to Put in Writing Studio

### Example 1: Professional Email
**Type this in the input box:**
```
Meeting with the design team next Tuesday to discuss the new app interface
```
**Then click:** "Professional Email" template

**Output:** A polished business email ready to send!

## Tips for Best Results

### ✅ DO:
- Be specific about what you want
- Provide context (who, what, when, why)
- Use templates for structured content
- Try different tones for variety
- Give details in your prompts

### ❌ DON'T:
- Use single words (be descriptive!)
- Expect perfect results every time
- Forget to select the right tone
- Skip reading the generated output
- Wait - start typing and try it!

## First Time Setup

1. **Open the app** - Wait 30-60 seconds for model download
2. **See "AI Ready" banner** - Green banner = ready to use!
3. **Go to Writing Studio** - Click examples or type your own
4. **Click any template** - Watch AI generate in real-time!
5. **Copy the output** - Use the Copy button

---

## Troubleshooting

**Q: Nothing happens when I click a template?**
- Make sure you typed something in the input box first!
- Wait for the "AI Ready" green banner

**Q: Output is not what I expected?**
- Try being more specific in your input
- Change the tone (try "Creative" or "Friendly")
- Click "Generate" again for a different result

**Q: Model keeps downloading?**
- This is now fixed! Model caches after first download
- Close and reopen the browser if issue persists

**Q: Image error in console?**
- This is now suppressed! The text model doesn't use images
- You can safely ignore any remaining image warnings
