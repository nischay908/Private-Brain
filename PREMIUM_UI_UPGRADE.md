## 📋 Still TODO

### Backend for Chat History (Requires Setup)

**Options:**
1. **LocalStorage** (Simplest - No backend needed)
   - Store in browser
   - Persists locally
   - Quick to implement
   - Works immediately

2. **IndexedDB** (Better for large data)
   - Browser database
   - More storage
   - Still no backend
   - Privacy maintained

3. **Firebase** (Cloud option)
   - Real backend
   - Sync across devices
   - Requires account setup
   - Costs money

4. **Supabase** (Open source alternative)
   - Similar to Firebase
   - PostgreSQL database
   - Free tier available
   - More control

**Recommendation:** Start with LocalStorage for simplicity and privacy-first approach. No backend setup needed!
