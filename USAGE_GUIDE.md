# ThinkLocal Usage Guide

## Getting Started

### First Launch
1. Open the app in your browser (Chrome/Edge recommended)
2. Wait for the AI model to download (~250MB, one-time only)
3. Model loads into memory (10-30 seconds)
4. You'll see a green "AI Ready" banner when ready!

### Navigation
Use the three tabs at the top to switch between features:
- **Writing Studio** - AI-powered writing assistant
- **Doc Analyzer** - Analyze and transform documents
- **AI Chat** - Have conversations with AI

---

## Writing Studio

### How to Use Templates
1. **Enter your topic or text** in the large text box
2. **Choose a tone** (Professional, Friendly, Formal, etc.)
3. **Click any template button** to generate:
   - **Professional Email** - Business-ready emails
   - **Blog Post** - Structured articles with intro/conclusion
   - **Bullet Summary** - Quick 5-7 point summaries
   - **Rewrite & Polish** - Improve clarity and flow
   - **Brainstorm Ideas** - Generate creative concepts
   - **Resume Bullet** - STAR method bullet points
   - **Explain Simply** - ELI5-style explanations
   - **Study Notes** - Structured learning materials

### Custom Generation
1. Type any request in the input box
2. Select your desired tone
3. Click **"Generate"** button
4. Watch as AI writes in real-time!

### Tips
- Use the **Stop** button to interrupt generation
- Click **Copy** to copy output to clipboard
- Word/character counts update in real-time
- Change tone anytime for different styles

---

## Document Analyzer

### How to Analyze Documents
1. **Paste your text** into the document box (articles, essays, reports, etc.)
2. **See instant stats**: words, read time, paragraphs, characters
3. **Choose an analysis mode**:
   - **Summarize** - 3-4 sentence overview
   - **Key Points** - 5 most important ideas
   - **Simplify** - Plain language version
   - **Critique** - 3 strengths + 3 improvements
   - **Generate Q&A** - 5 study questions
   - **Make Formal** - Academic/professional version
4. Click the analysis button and watch it work!

### Best Practices
- Paste up to ~2000 words for best results
- Use **Clear** button to start fresh
- Copy results for use elsewhere
- Try different modes on the same text

## AI Chat

### Quick Start Conversations
Click any starter prompt to begin:
- "Help me plan my week effectively"
- "Explain quantum computing simply"
- "Give me 5 productivity tips"
- "How do I improve my writing?"
- "What makes a great presentation?"
- "Help me brainstorm a startup idea"

### Having Conversations
1. Type your message in the input box
2. Press **Enter** to send (Shift+Enter for new lines)
3. Watch AI respond in real-time
4. Continue the conversation naturally
5. Use **Clear** to start a new topic

### Tips
- AI remembers last 3 exchanges for context
- Be specific in your questions
- All conversations stay on your device
- Chat history clears on page refresh

## Privacy & Performance

### Your Data is Safe
- **100% Private**: All AI runs in YOUR browser
- **No Cloud**: Zero data sent to any server
- **Offline**: Works without internet (after initial download)
- **No Tracking**: No analytics, no cookies, no accounts

### Performance Tips
- First generation may be slower (model warm-up)
- Close other tabs if memory is limited
- Chrome/Edge 120+ recommended for best speed
- Model stays cached - no re-download needed!

### Troubleshooting
- **Model won't download**: Check internet connection, try refreshing
- **Slow generation**: Close other browser tabs to free RAM
- **Errors**: Refresh the page to restart the SDK
- **Button not working**: Wait for "AI Ready" banner first

## Technical Details

**Model**: LFM2 350M (quantized GGUF)  
**Engine**: llama.cpp (WebAssembly)  
**Storage**: Origin Private File System (OPFS)  
**Generation**: ~10-30 tokens/second  
**Memory**: ~300-500MB RAM usage
